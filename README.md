# moments-replayed
A small text-based adventure game built in Java. Includes simple choices, a rewind feature, and multiple endings. Made to practice OOP and file handling concepts.

# Project Status
This project is currently in the early development stage. 
The intial setup and planning are complete - core gameplay and story logic will be implemented next.

# Planned Features
- Choice-based story with multiple endings
- Time rewind mechanic (undo the last decision)
- Simple Java console interface
- Object-oriented structure using Model-View-Controller (MVC)

# Development Roadmap
| Stage | Description |
|--------|--------------|
| **1. Setup** | Create project folders and base classes |
| **2. Core Loop** | Implement player input, scene transitions, and rewind |
| **3. Story Content** | Add branching storylines and endings |
| **4. Enhancements** | Add save/load, inventory, and polish |
| **5. Documentation** | Finalize README and comments for submission |
